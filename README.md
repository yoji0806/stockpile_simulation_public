このプロジェクトは、市民の防災水準の向上が目的です！  
大地震まで待ったなしの状況ですが、約50%の企業と世帯で災害用備蓄が不足しています。  
その原因の1つは、政府による画一的な情報提供です。  
開発している防災シミュレーターを用いて、1つ1つの企業や世帯に個別最適な情報を提供することで、市民全体の防災水準の向上を目指します。
